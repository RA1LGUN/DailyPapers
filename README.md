## 任务说明

### 1. 每日论文分析 (daily-papers)

**运行时间**: 每天 07:50 UTC

**任务内容**:
- 抓取 Hugging Face Papers
- 筛选符合研究兴趣的论文（机构、作者、方向）
- 提供深度分析（动机、假设、技术贡献、实验设计、评价）
- 严格、专业的数学化描述
- 附加 Quick Summary 轻量摘要，方便快速浏览

**研究偏好**:
- RL 演进：RLHF → DPO → GRPO → RLVR
- 数学推导与理论分析
- 数据构建、合成数据、筛选方法
- Benchmark、Code、Agent
- LLM training/scaling
- 范式转变

**筛选标准**:
- 机构：OpenAI, Anthropic, DeepSeek, Google DeepMind, Meta AI, 字节跳动, 阿里, 腾讯
- 作者：Kaiming He, Yann LeCun, Ilya Sutskever, Dario Amodei, Jan Leike, Josh Achiam 等
- 方向：LLM training/scaling, RLHF/DPO/GRPO/RLVR, Agent/Tool use, Reasoning/CoT/Math, Long context, MoE, Data efficiency/synthesis, Evaluation benchmarks, Embodied AI, Code generation

**标签说明**:
- `[OpenAI]` / `[Anthropic]` / `[DeepSeek]` / `[DeepMind]` / `[字节]` - 来源机构
- `[顶级学者]` - 核心作者
- `[强推]` - 强烈推荐
- `[有趣但有缺陷]` - 有趣但存在问题
- `[范式转变]` - 可能改变领域发展
- `[需验证]` - 需要进一步验证

**任务内容**:

- 抓取所有博客过去一周文章
- 筛选符合用户兴趣的文章
- 提供：来源、标题、发布日期、URL、内容摘要、评价
